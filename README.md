# AI WorkOS Academy

AI WorkOS Academy is an interactive, mission-based training platform that teaches learners how to use ChatGPT to complete real work outcomes (not just write better prompts).

## Product Principle

Learners train in a simulated workplace using a repeatable execution loop:

**COLLECT → REASON → EXECUTE → AUTOMATE**

## What is included in this repository

- Product specification for the tool
- Mission schema for content authoring
- GPT role prompts for assistant/simulator/evaluator/coach services
- Three production-ready Manager track missions
- API contract and scoring logic documentation

## Core Differentiator

Unlike prompt-only products, this platform scores end-to-end mission outcomes:

- Decision quality
- Communication quality
- Risk handling
- Time efficiency
- Automation readiness

## MVP Scope

1. Mission runtime engine
2. Workspace simulator (Inbox, Meetings, Docs, Data, Chat)
3. Agent orchestration panel
4. Deliverable desk (artifact bundles)
5. Evaluator and coaching loop
6. Progress unlocks

## Repository structure

- `docs/architecture.md` — architecture and system design
- `docs/api.md` — API contracts
- `schemas/mission.schema.json` — mission definition schema
- `prompts/*.md` — system prompts for model services
- `missions/manager/*.json` — authored missions
